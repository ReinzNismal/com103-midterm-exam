This Python script functions as a project management tracker that enforces specific input validation for the project title and group name. Once the user provides the correct credentials, the program displays a menu of task types and allows for four individual assignments to be created. For each assignment, the user designates a team member and a status, which the system then converts into a point value to measure progress. Finally, the script calculates a completion percentage and generates a Project Board summary that categorizes the project's overall readiness based on the total points earned.
